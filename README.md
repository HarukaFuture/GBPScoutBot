## 运行
`npm install`
`node bot.js`
备注:此程序需要配置才可运行
## 配置

	{
		"apikey":"[Your Telegram BotAPI Token]",
		"admin":[(管理员userid),(管理员userid)....],
		"gameserver":"[下文会提到]"
	}
### gameserver
gameserver的值:

- jp:日服
- kr:韩服
- tw:台服
- en:国际服
